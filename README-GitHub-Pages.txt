Pasos para publicar GRATIS con GitHub Pages

1) Crea una cuenta en https://github.com y verifica tu correo.
2) Crea un repositorio nuevo (botón "New") con el nombre que prefieras, por ejemplo `ingenioambiental-web`.
3) Sube los archivos `index.html` y `logo-iac.jpeg` a la raíz del repositorio.
4) Ve a Settings → Pages → "Deploy from a branch". Selecciona la rama `main` y la carpeta `/root`. Guarda.
5) Espera 1–2 minutos y abre la URL que te muestra GitHub Pages, será algo como:
   https://TU-USUARIO.github.io/ingenioambiental-web/

Opcional (mejora SEO y WhatsApp):
- En Settings → General, pon una descripción del repositorio.
- En la página, edita los textos "title" y "description" (ya vienen puestos).
- Comparte la URL por WhatsApp. Para un botón directo: https://wa.me/573108531744
- Si luego compras un dominio (p. ej., ingenioambiental.com.co), podrás conectarlo desde Settings → Pages → Custom domain.
